# Staging Deployment Repo

This repository is an artifact-only GitHub Pages deploy target for staging.cimc.ai.

- Do not develop features here
- Do not open source-code PRs here
- Do not edit files manually unless you are performing Pages or repository administration

Source code lives in the private repository:

- cimcai/website-main

Deployments are pushed automatically from the private source repo.

This repo should contain only built static site output and hosting files such as:

- index.html
- assets/
- CNAME
- .nojekyll
- README.md
